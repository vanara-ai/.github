# Vanara.ai

**Use AI to save time for what matters.**

We build open-source AI agents that take the grind out of the
repetitive, high-stakes work you do alone, starting with the job
search, expanding everywhere the same pattern shows up. Everything
is BYOK (bring your own key), MIT-licensed, and auditable end-to-end.
Your keys and your data never touch our servers.

## What we build

### [vanara-server](https://github.com/vanara-ai/vanara-server)

Python backend for AI resume optimization. Five-agent LangGraph pipeline,
FastAPI, fully BYOK. Your Groq API key is forwarded for the request and
immediately discarded — never logged, never cached.

### [vanara-ui](https://github.com/vanara-ai/vanara-ui)

Next.js 15 frontend. Your API key lives in your browser's localStorage
only. Pairs with `vanara-server` to deliver the full resume optimizer.

## What's next

More open-source agents for the work that eats your evenings. Same
rules every time: BYOK, MIT, your data stays yours.
